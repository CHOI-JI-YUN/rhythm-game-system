# 🎵 Rhythm Game Project

![Language](https://img.shields.io/badge/Language-C%2B%2B-blue)
![Library](https://img.shields.io/badge/Library-SFML-green)

C++ 기반 리듬 게임 및 미니게임 프로젝트

---

## 📌 프로젝트 소개
C++ 기반으로 다양한 미니게임과 리듬 게임 기능을 구현한 프로젝트입니다.

리듬 판정 시스템, 난이도 선택, 키 입력 처리, 점수 계산 등을 구현했으며,
팀원들과 단계별 게임을 연결하여 하나의 게임 흐름으로 구성했습니다.

---

## 📅 프로젝트 정보
- 개발 기간: 2026.03.30 ~ 2026.04.03
- 개발 형태: 팀 프로젝트
- 개발 언어: C++
- 개발 환경: Visual Studio

## 🎮 주요 기능
- 방향키 기반 리듬 입력 처리
- PERFECT / GOOD / MISS 판정 시스템
- 난이도 선택 기능
- 점수 및 콤보 시스템
- 미니게임 스테이지 구성
- 게임 오버 및 결과 화면 출력
